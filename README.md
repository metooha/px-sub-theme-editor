# PX Sub-Theme Editor

Living Design 3.5 theme editor and component library built with React and Vite.

## Development

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

## GitHub Pages

The site deploys automatically to GitHub Pages on every push to `main`.

Live URL: `https://<your-username>.github.io/px-sub-theme-editor/`
